View Project

View Project

This project aims to show the relationships between investors and ride-hailing companies in various countries and markets around the world.

It aims to give an overall view of the intertwined relationships, and a bit of background info, before diving into the scenarios in specific countries.

I gathered all of the investment data from Crunchbase and manipulated it using Python and Javascript. All of the data visualizations, the sankey diagram and country tables, are drawn using D3.
